# /slide - Marpスライド生成コマンド

**説明**: Marp形式のプレゼンテーションスライドを自動生成します。セミナー資料、ワークショップ資料、製品紹介スライドを、ビジュアル階層と一貫性のあるデザインで作成します。

## 使用方法

```
/slide [スライドの内容やトピック]
```

## 概要

`/slide` コマンドは、Slideスキルを呼び出し、Marp形式のプレゼンテーションスライドを自動生成します。用途に応じた3種類のテンプレート（一般登壇・セミナー、ハンズオンワークショップ、製品紹介）から適切なものを選択し、ビジュアル階層とデザインガイドラインに準拠したスライドを作成します。

## いつ使うか

- **技術カンファレンスやセミナーの登壇資料**: 問題提起から解決策、事例紹介までの流れ
- **ハンズオン形式のワークショップ**: 実習セクションとステップバイステップの説明
- **製品やサービスの紹介**: 課題、ソリューション、機能、デモの流れ
- **Marp形式でプレゼンテーション資料を作成**: 一貫性のあるデザインが必要な場合

## スライドの活用方法

### 1. プレビュー

生成されたスライドをMarp CLIでプレビューします：

```bash
# Marp CLIをインストール（初回のみ）
npm install -g @marp-team/marp-cli

# プレビュー
marp ./slides/your-slide.md

# PDFに変換
marp ./slides/your-slide.md --pdf

# HTMLに変換
marp ./slides/your-slide.md --html
```

### 2. カスタマイズ

生成されたスライドは、テンプレートをベースにしています。必要に応じて編集してください：

- フロントマターでスタイルを調整
- セクションを追加・削除
- 絵文字やビジュアル要素を変更

### 3. 共有

スライドを共有する方法：

- **PDF**: `marp --pdf` でPDFに変換して配布
- **HTML**: `marp --html` でWebページとして公開
- **Markdown**: そのまま共有（Marpがインストールされている環境で表示）

## テンプレートの選択基準

| テンプレート | 用途 | キーワード |
|-------------|------|-----------|
| **general-presentation** | 一般登壇・セミナー | "セミナー", "登壇", "カンファレンス", "勉強会" |
| **hands-on-workshop** | ハンズオンワークショップ | "ハンズオン", "実習", "ワークショップ", "workshop" |
| **product-intro** | 製品・サービス紹介 | "製品", "サービス", "ツール紹介", "product" |

## ベストプラクティス

1. **明確な要件**: トピック、対象者、時間を明確に伝える
2. **テンプレートの活用**: 適切なテンプレートを選択することで、効率的にスライドを作成
3. **カスタマイズ**: 生成後、必要に応じて編集
4. **プレビュー**: Marp CLIでプレビューし、デザインを確認
5. **一貫性**: ビジュアル階層とデザインガイドラインを維持

## リファレンスドキュメント

スライドをカスタマイズする際は、以下のリファレンスを参照してください：

- **visual-hierarchy.md**: ビジュアル階層ガイドライン（フォントサイズ、カラー、スペーシング）
- **emoji-usage.md**: 絵文字使用ガイド（カテゴリ別絵文字一覧、ベストプラクティス）
- **marp-syntax.md**: Marp構文リファレンス（フロントマター、スライド区切り、レイアウト）

パス: `skills/slide/references/`

## 関連スキル

- **doc-updater**: スライドのドキュメント更新

## モデル選択

スライド生成には、バランスの取れた**Sonnet**モデルの使用を推奨します。複雑なトピックや専門的な内容の場合は、**Opus**モデルを使用してください。

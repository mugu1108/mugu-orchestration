# /slide - Marpスライド生成コマンド

**説明**: Marp形式のプレゼンテーションスライドを自動生成します。

## 使用方法

```
/slide [スライドの内容やトピック]
```

## 概要

`/slide` コマンドは、Slideスキルを呼び出し、Marp形式のプレゼンテーションスライドを生成します。用途に応じたテンプレート（一般登壇・セミナー、ハンズオンワークショップ、製品紹介）を選択し、`skills/slide/references/` のガイドラインに準拠したデザインで作成します。

## いつ使うか

- **技術カンファレンスやセミナーの登壇資料**: 問題提起から解決策、事例紹介までの流れ
- **ハンズオン形式のワークショップ**: 実習セクションとステップバイステップの説明
- **製品やサービスの紹介**: 課題、ソリューション、機能、デモの流れ

## 使用例

### 例1: セミナー資料の作成

```
/slide Next.js 15の新機能についてのセミナー資料を作成してください。対象は中級者で、時間は30分です。
```

**出力**:
```markdown
# スライド生成結果

## サマリー
- テンプレート: general-presentation
- 対象者: 中級者 / 時間: 30分
- 生成枚数: 25枚

## 保存先
./slides/nextjs-15-features-20260227.md

プレビュー: npx @marp-team/marp-cli ./slides/nextjs-15-features-20260227.md
```

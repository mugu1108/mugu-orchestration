# Slide Creator Agent - スライド生成エージェント

## 概要

Slide Creatorエージェントは、Marp形式のプレゼンテーションスライドを自動生成する専門エージェントです。セミナー資料、ワークショップ資料、製品紹介スライドを、ビジュアル階層と一貫性のあるデザインで作成します。

## 使用タイミング

以下の場合にSlide Creatorエージェントを使用してください：

1. **技術カンファレンスやセミナーの登壇資料**
   - 問題提起から解決策、事例紹介までの流れ

2. **ハンズオン形式のワークショップ**
   - 実習セクションとステップバイステップの説明

3. **製品やサービスの紹介**
   - 課題、ソリューション、機能、デモの流れ

4. **Marp形式でプレゼンテーション資料を作成**
   - 一貫性のあるデザインが必要な場合

## ワークフロー

```
【スライド生成フロー】

ユーザーリクエスト
    │
    ↓
Slide Creator エージェント
    │
    ├── 1. 要件収集
    │   ├── テンプレートタイプ選択
    │   ├── 対象者・時間確認
    │   └── トピック・学習目標確認
    │
    ├── 2. テンプレート選択
    │   ├── general-presentation（一般登壇・セミナー）
    │   ├── hands-on-workshop（ハンズオン）
    │   └── product-intro（製品紹介）
    │
    ├── 3. アウトライン作成
    │   └── スライド構成の決定
    │
    ├── 4. Marpマークダウン生成
    │   ├── フロントマター設定
    │   ├── コンテンツ生成
    │   └── 絵文字・ビジュアル要素追加
    │
    ├── 5. 整合性チェック（内部）
    │   ├── コードブロック8行以内
    │   ├── コンテンツ量チェック
    │   └── Marp構文検証
    │
    ├── 6. ファイル保存
    │   └── ./slides/{topic}-{date}.md
    │
    └── 7. slide-reviewer サブエージェント呼び出し（オプション）
        ├── フォーマットレビュー
        ├── 内容レビュー
        └── 修正提案・自動修正
```

## テンプレート選択基準

| テンプレート | 用途 | キーワード |
|-------------|------|-----------|
| **general-presentation** | 一般登壇・セミナー | "セミナー", "登壇", "カンファレンス", "勉強会" |
| **hands-on-workshop** | ハンズオンワークショップ | "ハンズオン", "実習", "ワークショップ", "workshop" |
| **product-intro** | 製品・サービス紹介 | "製品", "サービス", "ツール紹介", "product" |

## コンテンツ制限ルール（必須）

### コードブロックの制限

| パターン | コード行数上限 | テキスト上限 |
|---------|--------------|-------------|
| コードのみ | **8行** | なし |
| コード + 説明 | **6行** | 2-3行 |
| コード + タイトル + ポイント | **6行** | ポイント2項目まで |
| 複数コードブロック | **合計8行** | なし |

**9行以上のコードは必ず分割**

### テキストのみの制限

| コンテンツタイプ | 上限 |
|----------------|------|
| 箇条書き（1階層） | **6項目** |
| 箇条書き（2階層） | **4項目**（サブ項目含む） |
| 段落 | **2-3段落** |
| テーブル | **5行 × 4列** |

## サブエージェント

### slide-reviewer

スライド生成後のレビュー・ブラッシュアップを担当するサブエージェントです。

**呼び出しタイミング**:
- スライド生成完了後（自動または手動）
- 既存スライドの改善時
- プレゼン前の最終チェック時

詳細は `agents/slide-reviewer.md` を参照。

## 使用例

### 例1: セミナー資料の作成

```
ユーザー: Next.js 15の新機能についてのセミナー資料を作成してください。
         対象は中級者で、時間は30分です。

Slide Creator:
1. 要件収集 → general-presentation, 中級者, 30分, Next.js 15
2. テンプレート選択 → general-presentation.md
3. アウトライン作成 → 25枚のスライド構成
4. Marpマークダウン生成 → コード8行以内、はみ出し防止
5. 整合性チェック → 全チェック項目クリア
6. ファイル保存 → ./slides/nextjs-15-features-{date}.md
7. レビュー → slide-reviewerでブラッシュアップ（オプション）
```

### 例2: ハンズオンワークショップ

```
ユーザー: TypeScript入門のワークショップ資料を作成してください。
         初心者向けで、2時間のハンズオン形式です。

Slide Creator:
1. 要件収集 → hands-on-workshop, 初心者, 2時間, TypeScript入門
2. テンプレート選択 → hands-on-workshop.md
3. アウトライン作成 → 40枚のスライド構成（実習5セクション）
4. Marpマークダウン生成 → 実習セクションを含む
5. 整合性チェック → 全チェック項目クリア
6. ファイル保存 → ./slides/typescript-intro-workshop-{date}.md
7. レビュー → slide-reviewerで初心者向け表現チェック
```

## リファレンスドキュメント

スライド生成時に参照するドキュメント：

| ファイル | 内容 |
|---------|------|
| `skills/slide/references/visual-hierarchy.md` | ビジュアル階層ガイドライン |
| `skills/slide/references/emoji-usage.md` | 絵文字使用ガイド |
| `skills/slide/references/marp-syntax.md` | Marp構文リファレンス |
| `docs/slide-content-guidelines.md` | コンテンツ量制限ルール |

## 関連ファイル

- `agents/slide-reviewer.md` - レビュー用サブエージェント
- `commands/slide.md` - コマンド定義
- `skills/slide/SKILL.md` - スキル定義（詳細ワークフロー）
- `skills/slide/templates/` - テンプレート一覧

## 出力先

```
デフォルト: ./slides/{topic}-{date}.md
例: ./slides/nextjs-15-features-20260129.md
```

## プレビュー方法

```bash
# プレビュー
npx @marp-team/marp-cli ./slides/your-slide.md

# PDFに変換
npx @marp-team/marp-cli ./slides/your-slide.md --pdf

# HTMLに変換
npx @marp-team/marp-cli ./slides/your-slide.md --html
```

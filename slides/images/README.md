# スライド用画像ディレクトリ

このディレクトリには、Marpスライドで使用する画像ファイルを配置します。

## プロフィール画像

自己紹介スライドで使用するプロフィール画像を `profile.png` という名前で配置してください。

### 推奨仕様
- ファイル名: `profile.png` または `profile.jpg`
- サイズ: 正方形（例: 400x400px、600x600px）
- フォーマット: PNG または JPG
- 背景: 透過または白背景推奨

## 使用方法

スライド内では以下のように画像を参照します:

```markdown
![w:200](images/profile.png)
```

- `w:200`: 幅200pxで表示（高さは自動調整）
- `images/profile.png`: このディレクトリからの相対パス

## その他の画像

スライドで使用するその他の画像もこのディレクトリに配置できます:
- スクリーンショット
- 図表
- ロゴ
- アイコン

ファイル名は分かりやすい名前にしてください（例: `architecture-diagram.png`, `demo-screenshot.png`）

## 実装済みエージェント

### 1. planner - 実装計画エージェント

**カテゴリ**: 開発支援
**コマンド**: `/plan [機能の説明]`
**推奨モデル**: Opus

#### 概要
複雑な機能や実装タスクの詳細な計画を作成する専門エージェント。実装に入る前に、要件分析、アーキテクチャ設計、ステップ分解を行う。

#### 使用タイミング
- 複雑な新機能の実装
- 大規模なリファクタリング
- 外部APIやサービスの統合
- セキュリティやパフォーマンスの改善

#### 詳細
[agents/planner.md](../agents/planner.md)

---

### 2. code-reviewer - コード品質レビューエージェント

**カテゴリ**: 開発支援
**コマンド**: `/code-review [ファイルパス]`
**推奨モデル**: Sonnet

#### 概要
コードの品質、可読性、保守性を自動的にレビュー。ベストプラクティスへの準拠、潜在的なバグ、パフォーマンス問題を検出。

#### レビュー項目
- コードスタイル（フォーマット、命名規則）
- ベストプラクティス（関数型、エラーハンドリング）
- パフォーマンス（アルゴリズム効率、DB）
- 可読性と保守性

#### 詳細
[agents/code-reviewer.md](../agents/code-reviewer.md)

---

### 3. security-reviewer - セキュリティレビューエージェント

**カテゴリ**: 開発支援
**コマンド**: `/security-review`
**推奨モデル**: Sonnet / Opus

#### 概要
コードのセキュリティ脆弱性を専門的に分析。OWASP Top 10に基づいた包括的なセキュリティチェックを実施。

#### チェック項目
- 認証と認可（RLSポリシー、JWT検証）
- インジェクション攻撃（SQL、コマンド）
- XSS（クロスサイトスクリプティング）
- 機密情報の漏洩（API Key管理）
- 依存関係のセキュリティ

#### 詳細
[agents/security-reviewer.md](../agents/security-reviewer.md)

### 5. time-tracker - 時間追跡エージェント

**カテゴリ**: 業務効率化
**コマンド**: `/in [プロジェクト]`, `/out`, `/status`
**連携**: Supabase, Notion

#### 概要
業務委託の作業時間を正確に記録・管理。Slackを通じて作業の開始・終了を記録し、SupabaseとNotionにデータを同期。

#### 機能
- プロジェクト別の作業開始・終了記録
- 日次・週次・月次の作業時間集計
- リアルタイムステータス確認
- Notion自動同期

#### 使用例
```
09:00 @timebot /in プロジェクトA
      → ✅ 作業を開始しました

12:00 @timebot /out
      → ✅ 作業を終了しました（3時間0分）
      → 📊 本日の合計: 3時間0分
```

#### 詳細
[agents/time-tracker.md](../agents/time-tracker.md)

### 6. invoice-generator - 請求書生成エージェント

**カテゴリ**: 業務効率化
**親エージェント**: time-tracker
**コマンド**: `/invoice [YYYY-MM]`（手動実行時）

#### 概要
Time Trackerのサブエージェント。月末に自動で請求書を生成。月間の作業時間を集計し、Excel形式の請求書をSlackにアップロード。

#### 機能
- 月末自動実行（毎月末日23:00）
- 締め日通知（Slackに送信）
- プロジェクト別の月間集計
- Excel請求書生成
- 消費税自動計算

#### 処理フロー
```
月末日 23:00
  ↓ 月末判定
  ↓ 作業データ取得（Supabase）
  ↓ 締め日通知（Slack）
  ↓ 請求書生成（Excel）
  ↓ Slackにアップロード
```

#### 詳細
[agents/invoice-generator.md](../agents/invoice-generator.md)

## 採用決定エージェント

### 7. デイリーブリーフィング（daily-briefing）

**カテゴリ**: 生産性
**ステータス**: 🚀 採用決定
**実装難易度**: 低

#### 概要
毎朝の情報をまとめてSlackに自動投稿。1日の準備を効率化。

#### 機能
- 今日のタスク（優先順位付き）
- カレンダーの予定
- 昨日の積み残し
- 重要な通知（GitHub、Slack）
- 天気予報

#### Slackメッセージ例
```
☀️ おはようございます！1月28日（火）

📅 今日の予定
• 10:00-11:00 | チーム定例MTG
• 14:00-15:00 | クライアントMTG

✅ 今日のタスク
🔴 [高] クライアントMTG資料作成
🟡 [中] ブログ記事執筆
🟢 [低] 技術調査

⚠️ 昨日の積み残し (2件)
• コードレビュー対応
• 請求書作成

🔔 未読通知
• GitHub PR: 3件
• Slack DM: 1件

🌤️ 天気: 晴れ（最高15℃/最低8℃）
```

#### 連携予定
- Googleカレンダー API
- Notion タスクDB
- GitHub API
- Weather API

## 計画中エージェント

### 8. 経費管理（expense-tracker）

**カテゴリ**: データ分析
**ステータス**: 📝 計画中（ステイ）
**実装難易度**: 中

#### 概要
レシート・請求書から経費を自動記録。確定申告の準備を効率化。

#### 機能
- OCRでレシート読み取り（Claude Vision API）
- 自動カテゴリ分類
- 勘定科目の提案
- Supabaseに保存
- 確定申告用CSVエクスポート

#### ワークフロー
```
1. レシート画像アップロード（Slack）
2. OCR + 情報抽出（Claude Vision）
3. カテゴリ自動分類
4. Supabase保存
5. Slack通知（確認ボタン付き）
6. 月次集計・CSV出力
```

#### カテゴリ
- 交通費
- 食費（会議費）
- 通信費
- 書籍
- その他
